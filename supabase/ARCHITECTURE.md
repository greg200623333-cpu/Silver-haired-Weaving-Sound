# 适老化极简鉴权设计

## 核心理念
老人不应面对任何密码输入框、验证码、或复杂的登录流程。

## 流程

1. **子女端（监护人）**：用微信小程序标准授权登录（wx.login → openid），填写老人基本信息（昵称、出生年份），生成专属绑定二维码。
2. **老人端**：打开小程序 → 静默调用 `wx.login` 获取 openid → 后端 `upsert_elder_by_openid` 自动查找/创建老人账号 → 直接进入回忆录音室主页。老人全程无感知。
3. **设备共享**：如果子女手机给老人用，提供一个「切换长辈模式」入口，切换后以绑定的 elder_id 进入，同样免密。

## 技术关键点
- `profiles` 表用 `wechat_openid` 作为免密凭据，微信生态天然安全。
- `magic_tokens` 表支持非微信场景（如 H5 端），由监护人短信/扫码生成一次性令牌。
- RLS 策略确保监护人只能看自己绑定的老人数据，老人只看自己的。
